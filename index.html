Exercício de CSS com Media Queries

O que é responsividade?

Responsividade, ou design responsivo, é uma abordagem de design de interface gráfica do usuário, usada para criar conteúdo que se ajusta suavemente a vários tamanhos de tela.

Vantagens de um design responsivo:

  • Permite uniformidade, o que é ótimo para a experiência do usuário;
  • Muitas ferramentas usam responsividade, como o Wordpress, por exemplo;
  • Relativamente fácil e barato de implementar;
  • Promove um design mais voltado à simplicidade e ao minimalismo.

O que são Media Queries?

Media queries são uma técnica do CSS3 que permite alterar o estilo de um elemento com base em certas características do dispositivo em que está sendo exibido.
